# block explorer

## 개요

- 블록, 트랜잭션, 지갑 주소에 대한 모든 정보를 확인할 수 있다.

## stack

- HTML, CSS, JavaScript, React, Node.js, mySQL, MetaMask, Web3, Geth

## 최소 기능 구현

- 블록, 트랜잭션, 지갑 주소 정보 출력
- 토큰의 코인에 대한 가격은 개인판단

- 블록을 db에 저장해서 검색호출하는게 빠르다.
